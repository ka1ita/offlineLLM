# Инициализация проекта

## Тип 

Консольное приложение для Windows

## Основные функции

- экспорт моделей ollama на компьютере, подключенном к сети Интернет в архивы, содержащие файлы моделей с ее параметрами. Перечень моделей для экспорта должен загружаться из текстового файла.
- импорт моделей на другом компьютере, не подклчюнном к сети Интернет, из софрмированных архивов.
- утилита должна формировать текстовый файл с перечнем самых популярных моделей ollama (или всех) для его редактирования и отправки на экспорт.
- ollama установлена на компьютере, где используется утилита.

## Технологический стек

Powershell или другой язык програмиирования простой в установке на Windows
